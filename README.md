# Config files to make my life easier

## Terminals

We have two config files for two different terminals:
- `alacritty.toml` is for `alacritty`. Put this under `~/.config/alacritty`
- `kitty.conf` is for kitty. Put this under `~/.config/kitty`.
  - `kitty.desktop` should be put under `~/.local/share/applications`, and should be executable
